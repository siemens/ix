## Prerequisites

- npm >=8.x.x
- node >=16.16.x

## Installation & Usage

```sh
yarn install
```

```sh
yarn build
```

```sh
yarn start
```

```sh
yarn build --filter=<workspace-name>
```

```sh
yarn start --filter=<workspace-name>
```
